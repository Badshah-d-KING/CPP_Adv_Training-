Question based on strings
3. We are implementing a cipher. Each letter of alphabet gets replaced by another one. For example 'c' is replaced
   by 'Z' and 'u' is replaced by 'A'.
   Write a program that ask user to enter a message. Encrypt this message and display it. Then decrypt the message to display 
   origninal message. 
   You should have two strings. One message and another one is key. To encrypt you can replace the character at position n in
   alphabet with the character at position n in key. For decrypt you can replace the character at position n in key with 
   character at position in alphabet.
